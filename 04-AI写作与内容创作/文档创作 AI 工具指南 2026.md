## 八、快速开始指南

### 1. 安装 Opencode Skills
```bash
# 查看可用技能
opencode skills list

# 使用 docx 技能
opencode skill use docx
```

### 2. 安装 VS Code 扩展
1. 打开 VS Code
2. 扩展商店搜索 "Markdown All in One"
3. 点击安装
4. 可选安装 Foam 或 Dendron

### 3. 安装 CLI 工具
```bash
# 安装 Pandoc
brew install pandoc  # macOS
choco install pandoc  # Windows

# 安装 mdBook
cargo install mdbook
```

### 4. 配置工作流
```markdown
# 示例：使用 Opencode 创建文档

1. 使用 Opencode 规划结构
   opencode "帮我规划一份技术文档大纲，主题是..."

2. 使用 doc-coauthoring 细化内容
   opencode skill use doc-coauthoring

3. 导出为 Markdown
   opencode "导出为 Markdown 格式"

4. 在 VS Code 中编辑
   code document.md

5. 使用 Pandoc 转换为 PDF
   pandoc document.md -o document.pdf
```

## 九、总结

### 最佳免费组合
- **日常写作**：Opencode + VS Code + Markdown All in One
- **学术论文**：Opencode + opendraft + Pandoc
- **长篇创作**：Opencode + NovelForge + mdBook
- **知识库**：Opencode + Foam/Dendron

### 与 Opencode 最佳集成
1. **原生 Skills**：docx、doc-coauthoring、writing-skills
2. **VS Code 协同**：Markdown All in One、Foam、Dendron
3. **CLI 集成**：Pandoc、mdBook（通过脚本调用）
4. **API 调用**：writing-agent、WriteHERE

### 注意事项
- 敏感数据优先选择本地工具（WitNote、本地 Ollama）
- 学术写作使用 opendraft 的引用验证
- 长篇文档使用 NovelForge 的结构化管理
- 技术文档使用 mdBook 生成可搜索网站
