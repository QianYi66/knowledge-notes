## 安装与使用

### 1. 安装 OpenDraft

```bash
# 克隆仓库
git clone https://github.com/federicodeponte/opendraft.git
cd opendraft

# 安装依赖
pip install -r requirements.txt

# 可选：安装 PDF 支持
pip install opendraft[pdf]
```

### 2. 配置 API Key

创建 `.env` 文件：

```bash
GOOGLE_API_KEY=your-gemini-api-key
ELEVENLABS_API_KEY=your-elevenlabs-key  # 可选，用于音频摘要
```

获取免费 Gemini API Key：https://makersuite.google.com/app/apikey

### 3. 快速生成论文

```python
from engine.draft_generator import DraftGenerator

generator = DraftGenerator()
draft = generator.generate(
    topic="The Impact of AI on Academic Research",
    paper_type="master",  # research_paper, bachelor, master, phd
    language="zh"  # 支持中文
)

# 导出格式
draft.to_pdf("thesis.pdf")
draft.to_docx("thesis.docx")
draft.to_latex("thesis.tex")
```

### 4. CLI 使用

```bash
# 生成论文
opendraft "AI 对学术研究的影响" --paper-type master --language zh

# 生成研究概览（快速）
opendraft "神经网络在医疗中的应用" --expose

# 生成 TL;DR 摘要
opendraft tldr paper.pdf

# 生成音频摘要
opendraft digest paper.pdf --voice adam

# 修订草稿
opendraft revise ./output "让引言更长，添加更多背景"

# 获取研究数据
opendraft data worldbank NY.GDP.MKTP.CD --countries USA;CHN --start 2020 --end 2023
```

### 5. 输出示例

**生成内容包含**
- 完整章节结构
- 2 万 + 字内容
- 40+ 条真实引用
- PDF/Word/LaTeX 格式
- 参考文献列表（BibTeX）

**样例下载**
- [PDF 样例（60 页）](https://opendraft.xyz/examples/Why_Academic_Thesis_AI_Saves_The_World.pdf)

## 学术写作最佳实践

### 1. 使用 Opencode 规划

```markdown
# 论文结构模板

## 摘要（200-300 字）
- 研究背景
- 主要贡献
- 关键发现

## 引言（1000-1500 字）
- 问题定义
- 研究动机
- 主要贡献
- 论文结构

## 相关工作（2000-3000 字）
- 分类综述
- 关键论文
- 研究空白

## 方法（3000-5000 字）
- 问题形式化
-  proposed 方法
- 技术细节
- 理论分析

## 实验（3000-5000 字）
- 数据集
- 基线方法
- 实验设置
- 主要结果
- 消融实验

## 结论（500-800 字）
- 主要发现
- 局限性
- 未来工作
```

### 2. 引用管理

**工具组合**
- Zotero / Mendeley（文献管理）
- Google Scholar Alerts（最新论文）
- Connected Papers（论文图谱）
- Semantic Scholar（推荐系统）

**引用格式**
```bibtex
@article{vaswani2017attention,
  title={Attention is all you need},
  author={Vaswani, Ashish and Shazeer, Noam and Parmar, Niki and Uszkoreit, Jakob and Jones, Llion and Gomez, Aidan N and Kaiser, {\L}ukasz and Polosukhin, Illia},
  journal={Advances in neural information processing systems},
  volume={30},
  year={2017}
}
```

### 3. 避免 AI 幻觉

**验证清单**
- [ ] 所有引用都可追踪
- [ ] 所有数据都有来源
- [ ] 所有图表都可复现
- [ ] 所有结论都有支撑
- [ ] 所有方法都可实施

**工具辅助**
- OpenDraft：自动验证引用
- scite.ai：引用上下文检查
- Retraction Watch：撤稿论文检查

## 常见问题

### Q: OpenDraft 真的免费吗？
A: 是的，100% 免费开源。只需要支付 API 费用（Gemini 免费层足够使用）

### Q: 可以直接提交 OpenDraft 生成的论文吗？
A: 不建议。OpenDraft 生成的是**研究草稿**，需要：
- 人工审查和修改
- 添加个人见解
- 验证所有引用
- 遵守学校 AI 政策

### Q: 支持中文论文吗？
A: 支持！57+ 种语言，包括中文

### Q: 引用准确率如何？
A: OpenDraft 声称接近 100%（验证 2.5 亿 + 论文库），但仍建议人工检查

### Q: 如何与 Opencode 配合使用？
A: Opencode 负责规划和结构化，OpenDraft 负责生成和引用验证
