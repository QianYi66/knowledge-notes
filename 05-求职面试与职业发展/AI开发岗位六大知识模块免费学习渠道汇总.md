# 第1章 数据结构与算法
学习路线: 掌握一门语言基础 → 学习基本数据结构 → 掌握经典算法 → 刷题实战
📗 入门阶段
推荐先修：基础编程能力（任何语言）。目标：理解数组、链表、栈、队列、树、图等核心数据结构。
📘 进阶阶段
目标：掌握排序、搜索、图算法、动态规划，理解算法复杂度分析。
📕 面试实战
目标：通过大量刷题巩固算法思维，准备技术面试。建议每天2-3题。
# 第2章 编程语言基础

🐍 Python（AI开发核心语言）
⚡ C++ / CUDA（高性能计算）
🐳 容器化与部署
# 第3章 AI/ML核心知识

# 附录：推荐学习顺序
建议按以下顺序逐步推进学习，每完成一个阶段再进入下一阶段：
第1步：Python入门（MIT 6.0001）→ 数据结构与算法入门（CS61B/中文B站）
第2步：算法进阶（Princeton Algorithms）→ 配合LeetCode刷题 → 准备面试
第3步：机器学习（吴恩达）→ 深度学习（DeepLearning.AI）→ Kaggle实战
第4步：LLM专项：Prompt Engineering → HuggingFace Agents → RAG/Agent实战
第5步：工程化：Docker → K8s → MLOps → 分布式训练
第6步：项目积累：开源贡献 → HuggingFace Spaces展示 → GitHub Profile建设
难度说明：⭐入门 = 零基础可学  ⭐⭐进阶 = 需一定基础  ⭐⭐⭐实战 = 需综合运用知识
